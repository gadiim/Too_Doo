TodoItemComponent: Відображає окрему справу.
TagComponent: Компонент для вибору тегів.
PriorityComponent: Компонент для вибору пріоритету справи.
V   TodoListComponent: Відображає список справ. 
V   TodoFormComponent: Форма для створення та редагування справ.    
TodoDetailComponent: Відображає деталі окремої справи.

ProjectComponent: Відображає окремий проект.
ProjectListComponent: Відображає список проектів.
ProjectFormComponent: Форма для створення та редагування проектів.
LocalStorageComponent: Компонент для збереження та зчитування даних з файлу
    TodoSearchComponent: Компонент для пошуку справ за різними критеріями.
FilterComponent: Компонент для фільтрації справ.
