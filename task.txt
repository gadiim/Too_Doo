TodoListComponent: Відображає список справ.
TodoItemComponent: Відображає окрему справу.
TodoFormComponent: Форма для створення та редагування справ.
TodoDetailComponent: Відображає деталі окремої справи.
ProjectComponent: Відображає список проектів.
ProjectFormComponent: Форма для створення та редагування проектів.
ProjectDetailComponent: Відображає деталі окремого проекту.
SearchComponent: Компонент для пошуку справ за різними критеріями.
FilterComponent: Компонент для фільтрації справ за тегами, пріоритетом тощо.
PriorityComponent: Компонент для вибору пріоритету справи.
TagComponent: Компонент для вибору тегів.
